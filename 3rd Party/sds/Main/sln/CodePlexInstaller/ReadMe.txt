To build SDS installer:

1. Download and install latest WiX Redist. 
 http://wix.sourceforge.net/releases
2. Download and install Sandcastle tools 
 http://www.microsoft.com/downloads/details.aspx?familyid=E82EA71D-DA89-42EE-A715-696E3A4873B2&displaylang=en
3. Download and install Sandcastle help file builder
 http://shfb.codeplex.com/releases/view/29710#DownloadId=99676
4. Load CodePlexInstaller.sln to your Visual Studio and select "Any CPU" build platform.
5. Run build of solution
